## Сортировка Хоара
 
Описание: Если список пуст, то результатом также будет пустой список. Иначе выделяется голова (первый элемент) и хвост (список из оставшихся элементов, который может быть пустым). В этом случае результатом будет конкатенация (сращивание) отсортированного списка из всех элементов хвоста меньших головы, списка из самой головы и списка из всех элементов хвоста бо́льших либо равных голове.
